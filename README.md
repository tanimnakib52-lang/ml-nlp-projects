# ML & NLP Projects — ITBI CUET Training

This repository contains the machine learning and NLP projects I completed during the AI & ML with Python training at IT Business Incubator, CUET (Chittagong University of Engineering and Technology) from July 2025 to March 2026.

Training score: ML — 40/50 | NLP — 50/50

## Projects

### 1. Tweet Sentiment Analysis
NLP-based sentiment classifier trained on tweet data.
- Preprocessing: tokenization, stopword removal, lemmatization (NLTK)
- Feature extraction: TF-IDF vectorization
- Models: Logistic Regression, SVM, Random Forest
- Evaluation: accuracy, precision, recall, F1-score
- Tools: Python, NLTK, scikit-learn, Pandas, Matplotlib

### 2. ML Classification Models
Comparison of multiple classification algorithms on structured datasets.
- Models tested: SVM, Random Forest, Logistic Regression, KNN
- Hyperparameter tuning with GridSearchCV
- Feature engineering and selection
- Tools: Python, scikit-learn, Pandas, NumPy, Seaborn

### 3. Data Analysis & Visualization
Exploratory data analysis on real-world datasets.
- Data cleaning and preprocessing
- Statistical analysis
- Visualization: histograms, heatmaps, scatter plots
- Tools: Pandas, Matplotlib, Seaborn

## Tools & Libraries
